# Priority A Simple Wireframes

更新日: 2026-04-20  
用途: 優先度 A の主要3ページに対する実装前の簡易ワイヤー  
対象ページ: `トップ` / `個人向けトップ` / `法人向けトップ`

## 1. このファイルの役割

このファイルは、`DESIGN.md v1`、構成ラフ、主要コピー初稿、写真監査を前提に、  
ページの骨組みだけを固定するための簡易ワイヤーである。

ここでは

- セクション順
- 情報の強弱
- 画像の置きどころ
- CTA の位置

を確認し、視覚表現の細部や CSS の設計までは入らない。

## 2. 表記ルール

- `[T]` はテキスト主導ブロック
- `[P]` は写真ブロック
- `[CTA]` は主導線
- `[Proof]` は実績 / 声 / 信頼補強
- `[Link]` は詳細ページへの導線

- Header は全ページ共通でシンプルに保つ
- CTA は強い装飾で煽らず、主要 CTA だけを目立たせる
- セクション背景の切り替えは多用せず、余白でリズムを作る
- 法人向けだけ色を硬くしすぎない
