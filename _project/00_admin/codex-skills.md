# Codex Skills For This Project

## 結論

この案件では、現時点で **必須の専用 skill はありません**。  
主作業は、ローカル資料の精査、情報設計、コピー設計、フロントエンド実装です。

## このセッションで実際に必要な能力

### いま必要

- 情報設計
- ブランド整理
- コンテンツ設計
- フロントエンド実装
- ファイル整理

### 後で必要になる可能性が高い

- 画像整理とビジュアル反映
- デザイン実装
- フォームやCMS連携の設計

## 現在インストール済み skill の扱い

### `imagegen`

使う可能性あり。

用途:

- 新しいビジュアル素材が必要になった場合
- 写真やチラシを元にしたラスタービジュアル案を作る場合

現時点では未使用で問題ありません。

### `openai-docs`

通常は不要です。

使う条件:

- サイト内で OpenAI API や ChatGPT 連携を組み込む場合
- OpenAI公式仕様の確認が必要な場合

### `plugin-creator`

不要です。

### `skill-creator`

不要です。

### `skill-installer`

不要です。

## 実務判断

この案件では、`skill を増やすこと` よりも、以下の順で進める方が成果に直結します。

1. 事業とブランドの整理
2. サイト構造の確定
3. 原稿作成
4. 実装

必要になった時点でだけ `imagegen` を使う、という運用で十分です。
