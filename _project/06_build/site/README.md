# Site Build Workspace

ここが、最終的なホームページ実装コードの配置先です。

## 想定する中身

- フロントエンドアプリ
- 画像 / public アセット
- ページ実装
- スタイル
- 必要な設定ファイル

## いまはまだ空でよい理由

現時点では、まだ以下が未確定です。

- ブランド名の最終決定
- サイトマップ
- 各ページの役割
- 原稿

この4つが固まってから実装を始める方が、作り直しが少なくなります。

## 実装開始の条件

次の条件がそろったら、このフォルダで実装を始めます。

1. トップページの主メッセージ確定
2. 個人向け / 法人向けの導線確定
3. 必須ページ一覧の確定
4. 最低限の原稿確定
