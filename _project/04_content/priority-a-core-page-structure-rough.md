# Priority A Core Page Structure Rough

更新日: 2026-04-20  
用途: 優先度 A の中でも、最初に固めるべき 3 ページの構成ラフ  
対象ページ: `トップ` / `個人向けトップ` / `法人向けトップ`

## 1. このファイルの役割

このファイルは、`DESIGN.md v1` を前提にしたページ構成ラフである。  
ここで決めるのは、実装や見た目の細部ではなく、各ページで

- 誰に向けて
- 何を
- どの順で
- どの温度感で
- どの CTA に接続するか

を固定すること。

次工程では、このラフをもとに

1. 主要コピー初稿
2. 写真候補の整理
3. 簡易ワイヤー
4. 実装

へ進む。  
主要コピー初稿は `priority-a-core-page-copy-draft.md` を参照する。

## 2. 3ページ共通の前提

- ブランドの主語は `竹村英子`
- 第一印象は `整える専門家`
- BtoC 主軸、BtoB は第2の柱
- 個人向け主入口は `無料診断`
- 法人向け主入口は `相談 / 問い合わせ`
- サービス名の列挙ではなく、支援価値の理解を先に置く
- 余白を活かし、セクション数を増やしすぎない

## 3. トップページ構成ラフ

### 3-1. このページの役割

- 竹村英子さんが何者かを短時間で伝える
- 個人向け / 法人向けの入口を迷わず選ばせる
- `整える専門家` としての信頼を短時間で立てる

### 3-2. 主な読み手

- 初めて竹村英子さんを知る個人ユーザー
- 紹介や検索で来た法人担当者
- すでに関係性はあるが、改めて案内先を確認したい人

### 3-3. このページの到達点

- `個人の方へ` または `法人・団体の方へ` を選べる
- 竹村英子さんの支援領域を誤解なく理解できる
- 個人向けでは `無料診断` に進みやすくなる

### 3-4. 推奨セクション順

#### 1. ファーストビュー

目的:

- `心・体・習慣を整える専門家` であることをひと目で伝える
- 個人向け / 法人向けの二導線を早い段階で見せる

入れる内容:

- 竹村英子さん本人写真
- 主メッセージ
- 補足の短文
- `個人の方へ`
- `法人・団体の方へ`
- 補助CTAとしての `無料診断`

避けること:

- いきなりサービス一覧にする
- 実績や肩書きを詰め込みすぎる

#### 2. 導線セレクター

目的:

- 読み手が自分の入口を迷わず選べるようにする

入れる内容:

- 個人向け導線の要約
- 法人向け導線の要約
- 各導線の先にある価値の一言

メモ:

- ここはカード連打ではなく、余白のある比較導線が望ましい

#### 3. 竹村英子さんの支援価値

目的:

- `何をしてくれる人か` を方法論より先に理解させる

入れる内容:

- 整える対象
- 支援スタイル
- 個人 / 法人の両方に一貫する視点

メモ:

- `色` `食` `体` はここでは素材ではなく支援軸として簡潔に見せる

#### 4. 個人向け価値の導入

目的:

- BtoC 主軸であることを自然に見せる
- 個人向けページや `無料診断` に接続する

入れる内容:

- 40代以上女性に響く悩みの整理
- `整えられるものだ` という再定義
- `無料診断` CTA
- `個人の方へ` CTA

#### 5. 法人向け価値の導入

目的:

- BtoB を第2の柱として信頼感のある形で見せる

入れる内容:

- 企業人事が抱える課題の要約
- 研修 / 相談で支援できる領域
- `法人・団体の方へ` CTA

#### 6. 一貫性のある実績 / 信頼補強

目的:

- `何でも屋` ではなく `整える専門家` として信頼を補強する

入れる内容:

- 実績の数ではなく領域の一貫性
- 法人実績の要約
- 個人向けの声の要約
- 必要最小限の肩書き / 年数

#### 7. プロフィール導入

目的:

- 人柄と専門性の両方を感じさせる

入れる内容:

- 竹村英子さんの短いプロフィール
- `竹村英子について` への導線

#### 8. フッター前 CTA

目的:

- 読み終わった人が迷わず次へ進めるようにする

入れる内容:

- `個人の方へ`
- `無料診断`
- `法人・団体の方へ`
- `問い合わせ / 相談`

### 3-5. このページで必要になる素材

- 竹村英子さんの主写真
- 個人向け補助写真
- 法人向け補助写真
- 実績要約用の短い証拠素材

### 3-6. このページでまだ原稿化していないもの

- 主メッセージ
- 個人向け価値の短い要約
- 法人向け価値の短い要約
- 実績の見出し文

## 4. 個人向けトップ構成ラフ

### 4-1. このページの役割

- 40代以上女性が、自分ごととして読める入口を作る
- 不調や揺らぎを、過剰に不安化せず整理して受け止める
- `無料診断` と `じぶん整え習慣サロン` へ自然につなぐ

### 4-2. 主な読み手

- 心身の揺らぎを感じている 40 代以上女性
- 健康情報は見ているが、自分に合う整え方が定まっていない人
- すぐ申込むより、まず理解したい人

### 4-3. このページの到達点

- `これは自分向けの支援だ` と感じられる
- `無料診断` を気軽に試せる
- `じぶん整え習慣` と `サロン` に興味を持てる

### 4-4. 推奨セクション順

#### 1. ファーストビュー

目的:

- 自分の状態をわかってくれそうだと感じてもらう
- 強い煽りなく、安心して読み始められるようにする

入れる内容:

- 共感起点の主メッセージ
- 支援の要約
- 竹村英子さんの伴走感が伝わる写真
- `無料診断`

避けること:

- 病名や恐怖訴求
- 癒し系ワードの多用

#### 2. よくある揺らぎの整理

目的:

- 読み手の状態を言語化し、自分ごと化を強める

入れる内容:

- 体の揺らぎ
- 気持ちの揺らぎ
- 生活リズムの揺らぎ

メモ:

- 項目数を増やしすぎず、読みながら頷ける粒度にする

#### 3. `整える` という再定義

目的:

- 根性論や我慢ではなく、整え方の問題として捉え直してもらう

入れる内容:

- ただ頑張るのではなく、整える視点が必要であること
- 変化は習慣の積み上げで起こること

#### 4. `じぶん整え習慣` の考え方

目的:

- 個人向け支援の中心思想を理解してもらう

入れる内容:

- なぜこの支援をしているのか
- どんな人に合う考え方か
- `じぶん整え習慣とは` への導線

#### 5. 3つの支援軸

目的:

- 支援内容をふわっとさせず、わかりやすく整理する

入れる内容:

- `体養生`
- `食養生`
- `色養生`

メモ:

- ここは図解や短い説明で整理し、長文にしない

#### 6. `無料診断` の案内

目的:

- 主入口としての行動を起こしやすくする

入れる内容:

- 診断でわかること
- 診断後の流れ
- 売り込み前提ではない安心感
- `無料診断` CTA

#### 7. `じぶん整え習慣サロン` の導入

目的:

- 診断の先にある主力商品のイメージを自然に持ってもらう

入れる内容:

- どんな人のための場か
- 何が得られるか
- `動画101本` と 3 領域の要約
- `じぶん整え習慣サロン` への導線

#### 8. 受講者の安心材料

目的:

- 自分にも続けられそうだと感じてもらう

入れる内容:

- 個人向けの声
- 小さな変化の実感
- 継続しやすさ

#### 9. 竹村英子さんの人柄 / 背景

目的:

- 支援者として信頼しやすくする

入れる内容:

- プロフィール要約
- 支援の姿勢
- `竹村英子について` への導線

#### 10. フッター前 CTA

目的:

- 理解が進んだ人を次に送る

入れる内容:

- 主CTA: `無料診断`
- 補助CTA: `無料相談`
- 補助導線: `じぶん整え習慣サロン`

### 4-5. このページで必要になる素材

- 竹村英子さんの伴走感が伝わる写真
- 生活文脈の補助写真
- 受講者の声
- 診断後フローの説明素材

### 4-6. このページでまだ原稿化していないもの

- ファーストビューの共感コピー
- `整える` の再定義コピー
- 診断案内の安心文
- サロン導入の短い要約

## 5. 法人向けトップ構成ラフ

### 5-1. このページの役割

- 企業人事に `この人に相談できる` と伝える
- 研修や支援が何を改善するのかをわかりやすく示す
- 詳細ページや `相談 / 問い合わせ` へつなぐ

### 5-2. 主な読み手

- 企業人事
- 研修 / 健康経営 / 福利厚生の担当者
- 社員のコンディションや職場コミュニケーションに課題を感じる担当者

### 5-3. このページの到達点

- 課題に対して相談可能な支援者だと理解できる
- 提供できる支援の範囲を把握できる
- `相談 / 問い合わせ` または詳細ページに進める

### 5-4. 推奨セクション順

#### 1. ファーストビュー

目的:

- `職場の心身を整える支援` ができる専門家だと伝える
- 研修会社ではなく、相談できる専門家として見せる

入れる内容:

- 法人向けの主メッセージ
- 解決できる課題の要約
- 竹村英子さん本人写真
- `相談する`
- `研修メニューを見る`

避けること:

- 抽象的な理念だけで終わること
- 企業サイト風の無機質な見せ方

#### 2. よくある組織課題の整理

目的:

- 企業人事が自社課題に接続しやすくする

入れる内容:

- コミュニケーション課題
- ストレス / コンディション課題
- 健康経営 / 離職予防 / 関係性課題

#### 3. 竹村英子さんに相談できる理由

目的:

- 個人支援と法人支援の両方を持つ強みを伝える

入れる内容:

- 現場理解
- 対話型支援
- 心 / 体 / 習慣を横断して見られること

#### 4. 提供できる支援メニューの骨子

目的:

- `何が頼めるか` を明快にする

入れる内容:

- 職場コミュニケーション改善
- ストレス可視化とセルフケア
- 健康経営 / 心身コンディショニング
- 福利厚生型の継続支援

メモ:

- 数を増やしすぎず、主力を 3 から 4 本に絞る

#### 5. 導入イメージ / 支援の進め方

目的:

- 依頼後のイメージを持ってもらう

入れる内容:

- ヒアリング
- 課題整理
- 企画 / 実施
- フォロー

#### 6. 実績 / 導入事例の要約

目的:

- 安心材料を補強する

入れる内容:

- 法人実績
- 研修 / 講演の実施例
- 実名掲載の可否に応じた表現

#### 7. 継続支援の可能性

目的:

- 単発研修だけでなく、中長期支援も相談できる印象を作る

入れる内容:

- 福利厚生型の継続提供
- 伴走支援の余地

#### 8. プロフィール導入

目的:

- 支援者本人への信頼をつくる

入れる内容:

- 竹村英子さんのプロフィール要約
- `竹村英子について` への導線

#### 9. フッター前 CTA

目的:

- 相談アクションに迷いなく接続する

入れる内容:

- 主CTA: `相談する`
- 補助CTA: `問い合わせる`
- 補助導線: `研修メニューを見る`

### 5-5. このページで必要になる素材

- 法人向けの本人写真
- 研修 / 登壇の実績素材
- 導入事例の短い要約
- 問い合わせ導線の説明素材

### 5-6. このページでまだ原稿化していないもの

- 法人向けファーストビューの主メッセージ
- 課題整理の見出し群
- メニュー骨子の短い説明
- 相談 CTA の文言

2. 各セクションの見出し案
3. 各セクションの本文初稿
4. 必要写真と不足素材の一覧
