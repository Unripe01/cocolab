# 竹村英子ホームページ ページ別原稿設計書 v1

作成日: 2026-04-25  
更新日: 2026-04-25  
用途: デザイン・実装に渡すためのページ別原稿設計整理  
ステータス: 優先3ページ整理版

## 1. この資料の役割

この資料は、[site-structure-final-v1-for-hideko.md](./site-structure-final-v1-for-hideko.md) を前提に、  
次工程で必要になる `ページごとの役割` `主メッセージ` `セクション順` `CTA` `必要素材` を整理した handoff 用の資料です。

ここで固定したいのは、見た目の細部ではなく、

- 誰に向けるか
- 何を最初に伝えるか
- どの順で理解させるか
- 最後にどこへ送るか

です。

## 2. 先に結論

最初に固めるべきページは、次の 3 ページです。

1. `ホーム`
2. `個人の方へ`
3. `法人・団体の方へ`

理由は、この 3 ページがサイト全体の主語・導線・信頼感を決めるからです。  
他ページは、この 3 ページでできた文脈に沿って詳細化すれば進めやすくなります。

法人向けでは、`健康経営をやっています` のような広い名乗り方はしません。  
竹村英子さんの提供価値は、健康経営全体の統括ではなく、`心・体・習慣を一体で整える支援` です。

## 3. 優先3ページの共通ルール

- サイト全体の主語は `竹村英子`
- 第一印象は `整える専門家`
- BtoC 主軸、BtoB は第2の柱
- 個人向けの主入口は `無料診断`
- 法人向けの主入口は `相談 / 問い合わせ`
- サービス名の列挙より、支援価値の理解を先に置く
- 実績は `量` ではなく `一貫性` で見せる
- 法人向けでは `健康経営` を補助語として扱うことはあっても、主サービス名にはしない

## 4. ホーム

### 4-1. このページの役割

- 竹村英子さんが何者かを短時間で伝える
- 個人向け / 法人向けの入口を迷わず選ばせる
- `整える専門家` としての信頼を立てる

### 4-2. 主メッセージの方向

- 主コピー: `心・体・習慣を整える。`
- 補足コピー: `竹村英子は、女性の毎日と職場の健やかさに寄り添う整える専門家です。`
- 説明文: `個人の方には、じぶん整え習慣を。法人・団体の方には、職場の心身を整える支援をお届けします。`

### 4-3. 推奨セクション順

1. ファーストビュー
2. 個人向け / 法人向けの導線セレクター
3. `心・体・習慣` を整える支援価値の説明
4. 個人向け価値の導入
5. 法人向け価値の導入
6. 一貫性のある実績 / 信頼補強
7. プロフィール導入
8. フッター前 CTA

### 4-4. 主CTA

- `個人の方へ`
- `法人・団体の方へ`
- `まずは無料診断`

### 4-5. 必要素材

- 竹村英子さんの主写真
- 個人向け補助写真
- 法人向け補助写真
- 実績を短く示せる証拠素材

## 5. 個人の方へ

### 5-1. このページの役割

- 40代以上女性が、自分ごととして読める入口を作る
- 不調や揺らぎを過剰に不安化せず、整理して受け止められるようにする
- `無料診断` と `じぶん整え習慣サロン` へ自然につなぐ

### 5-2. 主メッセージの方向

- 主コピー: `なんとなくつらい毎日を、じぶんで整えられる毎日へ。`
- 補足コピー: `40代からの心と体の揺らぎを、我慢ではなく習慣で整えるサポートです。`
- 説明文: `竹村英子が、体・食・色の視点から、あなたに合った整え方を見つけるお手伝いをします。`

### 5-3. 推奨セクション順

1. ファーストビュー
2. よくある揺らぎの整理
3. `整える` という再定義
4. `じぶん整え習慣` の考え方
5. `体養生` `食養生` `色養生` の3軸
6. `無料診断` の案内
7. `じぶん整え習慣サロン` の導入
8. 続けやすさや受講者の安心材料
9. プロフィール導入
10. フッター前 CTA

### 5-4. 主CTA

- `無料診断を受ける`
- `じぶん整え習慣とは`
- `サロンを見る`

### 5-5. 必要素材

- 竹村英子さんの伴走感が伝わる写真
- 生活文脈の補助写真
- 受講者の声
- 診断後フローの説明素材
- サロンの説明に使える既存チラシや概要素材

## 6. 法人・団体の方へ

### 6-1. このページの役割

- 企業人事に `この人に相談できる` と伝える
- 何を改善できる支援かをわかりやすく示す
- `相談 / 問い合わせ` と `研修メニュー` に接続する

### 6-2. 主メッセージの方向

- 主コピー: `職場の心身を整える支援を、現場に合わせて。`
- 補足コピー: `コミュニケーション、ストレス、メンタル・身体・習慣づくりの課題に、竹村英子が寄り添って伴走します。`
- 説明文: `研修を行うだけでなく、現場の状態を整理しながら、職場に合った整え方をご提案します。`

### 6-3. 表現上の注意

- `健康経営をやっています` とは書かない
- `健康経営` は企業側の課題文脈で補助的に触れるのは可
- 前面に出す支援領域は `メンタル` `身体` `習慣化`
- 主力メニュー名は `心・体・習慣のコンディショニング` に揃える

### 6-4. 推奨セクション順

1. ファーストビュー
2. よくある組織課題の整理
3. 竹村英子さんに相談できる理由
4. 提供できる支援メニューの骨子
5. 導入イメージ / 支援の進め方
6. 実績 / 導入事例の要約
7. 継続支援の可能性
8. プロフィール導入
9. フッター前 CTA

### 6-5. 主CTA

- `相談する`
- `問い合わせる`
- `研修メニューを見る`

### 6-6. 必要素材

- 法人向けの本人写真
- 研修 / 登壇実績の証拠素材
- 導入事例の短い要約
- 問い合わせ導線の説明素材

## 7. 優先度Bのページ

以下は、優先3ページの文脈が固まったあとに詳細化するページです。

- `竹村英子について`
  プロフィール、活動実績、想い、カラーココロジー研究所、方法論紹介を整理する
- `じぶん整え習慣とは`
  考え方の説明ページとして使い、個人向けトップの理解を深める
- `じぶん整え習慣サロン`
  主力商品ページとして、対象者、内容、動画101本、参加導線を整理する
- `無料診断`
  何がわかるか、診断後の流れ、メール導線を明確にする
- `無料相談 / 個別相談`
  診断後の次アクションとして整理する
- `実績・お客様の声`
  個人向け、法人向け、教育 / 就職支援などを分けて見せる
- `お問い合わせ`
  個人向け相談か法人向け相談かを迷わず選べる入口にする

## 8. まだ必要な素材と確認事項

- `日本機設工業株式会社様` の実名掲載可否
- 法人向けで主役にしたい導入事例の選定
- 個人向け受講者の声の整理
- 法人向け実績の掲載粒度
- 竹村英子さんの写真素材の使い分け
- サロン参加導線で使う外部リンクの見せ方

## 9. 制作順の推奨

1. `ホーム` `個人の方へ` `法人・団体の方へ` の原稿方向を固定する
2. 写真・実績・声などの素材をページごとに割り当てる
3. ワイヤーまたはデザインに落とす
4. 優先3ページから実装する
5. その後に下層ページを詳細化する
