# generated-image-assets

このフォルダは、2026-04-26 に追加・整理した生成画像系の原本資料を保管する場所です。

## 内容

- `ヒーロー・背景用.png`: トップ・下層ページ背景の原本
- `心を整えるアイコン.png`: 個人向け価値カード用アイコン原本
- `体を整えるアイコン.png`: 個人向け価値カード用アイコン原本
- `習慣を整えるアイコン.png`: 個人向け価値カード用アイコン原本
- `企業用課題整理アイコン.png`: 法人向け課題・メニュー用アイコン原本
- `おすすめ画像一覧.png`: 参照用一覧画像
- `画像説明.md`: 画像整形・組み込み方針メモ

## 公開サイトで使う画像

公開HTML/CSSからは、このフォルダを直接参照しません。
実装用に整形した画像は `_project/06_build/site/assets/generated/` に配置しています。
