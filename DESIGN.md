# DESIGN.md

ステータス: v1 初期確定版  
更新日: 2026-04-20  
用途: 実装前のデザイン判断を固定するための制御ドキュメント

## 1. このファイルの役割

この `DESIGN.md` は、実装仕様書ではない。  
竹村英子ホームページ刷新において、ワイヤーや画面実装へ進む前に

- 何をどう見せるか
- どの温度感で伝えるか
- BtoC / BtoB をどう同居させるか
- どこまでを固定し、どこからを後工程に回すか

を固定するためのデザイン制御ドキュメントとして扱う。

今後、画面案・原稿・実装判断で迷った場合は、まずこのファイルに戻る。

## 2. この案件で確定している戦略前提

- サイトは `竹村英子` を主語にした 1 サイト構成
- BtoC を主軸、BtoB を第2の柱にする
- 個人向け主力商品は `じぶん整え習慣サロン`
- `カラーココロジー研究所` は屋号として残す
- `ごきげん体質` は今後使わない
- 個人向け主ターゲットは `40代以上女性`
- 法人向け主ターゲットは `企業人事`
- 個人向け主入口は `無料診断`
- 診断後はメールナーチャリングを通じて `無料相談` やサロン参加へ接続する
- サロン申込の外部LPは、いきなり見せず、説明や関係構築の後で案内する
- 実績は `量` ではなく `一貫性` で見せる

## 3. v1 で固定する結論

### 3-1. トンマナの主軸

主軸は `Calm Editorial` とする。

採用理由:

- `整っている`
- `信頼できる`
- `女性向けでも軽すぎない`
- `法人向けとの接続がしやすい`

という条件を、最も無理なく満たせるため。

ただし、冷たく見せないために

- 写真
- 色の副調
- コピーの語尾
- 余白の取り方

には `Warm Holistic` のやわらかさを補助的に入れる。

`Refined Natural` の要素は、自然物の写真や素材感で補助的に使う。  
サイト全体の見え方そのものをナチュラル寄りにしすぎない。

### 3-2. 一言でいうと何を目指すか

このサイトが目指す見え方は、

`整える専門家が、静かに信頼をつくりながら伴走してくれる`

である。

### 3-3. この v1 で確定した範囲

この v1 では、次を固定済みとする。

- トンマナの主軸
- 写真 / 色 / タイポ / 余白の方向
- BtoC / BtoB の見せ分けルール
- CTA の圧力設計
- トップ / 個人向け / 法人向けを中心にした画面方針

この段階では、まだ次は固定しない。

- CSS の詳細設計
- コンポーネント分解
- CMS の詳細設計
- micro interaction の細部

## 4. サイトが解決すべきこと

現行サイトでは、

- 何をしている人なのか
- 個人向けなのか法人向けなのか
- いま一番案内したいサービスが何なのか

が初見で掴みにくい。

新サイトでは、次を達成する必要がある。

1. 竹村英子さんの専門性を短時間で理解できる
2. 個人向け / 法人向けの導線が迷わず選べる
3. 個人向けでは `じぶん整え習慣サロン` へ自然に関心が向く
4. 法人向けでは `相談できる専門家` として信頼が立つ
5. 実績が `何でも屋` ではなく `整える専門家` として見える

## 5. 表現の主語

- ブランドの主語は `竹村英子`
- 屋号の `カラーココロジー研究所` は信頼資産として扱う
- `コミュニケア®` は方法論のひとつとして扱う
- サイト全体を、診断メソッドや講座名の集合体にはしない
- 公開本文の基本の語り手は、本人サイトとしての `私`
- 想い、支援姿勢、相談導線では、読み手に `あなた` / `ご担当者` と語りかける
- 経歴、資格、実績、ページタイトル、ナビゲーション、画像altでは、識別性と客観性のために `竹村英子` を使ってよい
- 制作会社や第三者紹介記事のような `竹村英子は...` `竹村英子さんの...` の連続は避ける

## 6. 守る印象と避ける見え方

### 6-1. 絶対に守る印象

- 整っている
- 信頼できる
- 押しつけがましくない
- 女性向けでも軽すぎない
- 専門家だが冷たくない
- 健康や心の話でもスピリチュアルに寄りすぎない

### 6-2. 避けるべき見え方

- 量産型の自己啓発サイト
- ふわっとしたスピリチュアル表現
- かわいさ優先で信頼が落ちる見え方
- 法人向けに寄せすぎて個人向けの親しみが消える見え方
- サービスを並べすぎて主力が見えない見え方
- 実装都合で均質化されたテンプレート感
- ストック写真だけで作られた無機質な wellness サイト

## 7. デザイン原則

1. まず `整える人` として見えること
2. `竹村英子` 本人の信頼を最前面に置くこと
3. 余白と情報整理で安心感を出すこと
4. BtoC は親密さ、BtoB は信頼感を出し分けること
5. ただし別サイトのように分断しないこと
6. CTA は強引に押さず、理解の流れの中で配置すること
7. 外部LPへの遷移は、説明後の納得導線として扱うこと
8. 実績は `量` より `今の価値につながる一貫性` で見せること
9. 方法論よりも、相手の変化や支援価値が先に伝わること

## 8. ビジュアルシステム方針

### 8-1. 写真 / ビジュアル方向

- 竹村英子さん本人写真を主役にする
- イラストは主軸にしない
- 顔写真だけでなく、話している、寄り添っている、伝えている文脈の写真を優先する
- BtoC では、表情、手元、暮らし、食、体を連想できる寄りの写真を補助的に使う
- BtoB では、登壇、対話、研修、場づくりが伝わる写真を優先する
- 色・食・体の要素は、説明用の象徴素材として使いすぎず、文脈を支える補助視覚として使う
- 抽象的すぎる自然写真、空、光、瞑想ポーズ、過度な癒し演出は避ける
- 法人向けで安いストック感のある握手写真や会議室写真は避ける

### 8-2. 色の方向

色は `落ち着き` と `体温` を両立させる。

- ベースは、白すぎないオフホワイト
- 補助ベースに、サンドベージュ、ストーングレー系を置く
- 文字色は、真っ黒ではなくインク感のあるダークトーンを使う
- 強い差し色は避け、セージ、深い青緑、くすんだローズ系を限定的に使う
- BtoC は少しあたたかく、BtoB は少し引き締める
- ただし、ページごとに別ブランドのような配色差は作らない

### 8-3. タイポグラフィ方向

- 見出しは、上質さと落ち着きを出せる明朝系またはセリフ系を軸にする
- 本文は、可読性が高く、冷たすぎないゴシック系を軸にする
- 1画面の中でフォント数を増やしすぎない
- 女性向けだからといって装飾的な手書き風には寄せない
- 数字、実績、法人向け情報は整ったサンセリフで見せる
- 行間はやや広めに取り、圧迫感を避ける

### 8-4. 余白 / 情報密度 / レイアウト

- 情報は詰め込みすぎず、誌面のように整理する
- ファーストビューは情報を欲張らず、役割をひと目で理解できる密度に抑える
- セクション間はしっかり余白を取り、呼吸できる画面にする
- カードの連打で埋めるのではなく、コピー、写真、実績ブロックの強弱でリズムを作る
- BtoC は少しやわらかく余白を多めに、BtoB は少し密度を上げて論理性を出す
- ただし、共通のグリッドと縦リズムは崩さない

### 8-5. 動きの扱い

- 動きは補助的に使う
- あるとしても、静かなフェード、段差の少ない出現、読み進めやすさを助ける程度に留める
- 派手なアニメーション、浮遊感の強い演出、癒し系の過剰な動きは避ける

## 9. BtoC / BtoB の見せ分けルール

### 9-1. 共通して守ること

- ブランド主語は常に `竹村英子`
- 同じ色体系、同じタイポ体系、同じ余白思想でつなぐ
- どちらの導線でも `整える専門家` という見え方を崩さない

### 9-2. BtoC で強めること

- 共感しやすい悩みの言語化
- やわらかい写真の寄り方
- 読み進めやすい余白
- 小さな安心の積み上げ
- `無料診断` への自然な入口

### 9-3. BtoB で強めること

- 相談できる専門家としての信頼
- 課題整理の明快さ
- 研修や支援内容の構造化
- 実績や導入イメージの整理
- `問い合わせ / 相談` への明確な導線

### 9-4. やってはいけないこと

- BtoC だけ丸みと淡色で構成し、BtoB だけ硬い企業サイトにすること
- 入口以降でブランド人格が切り替わること
- 個人向けで感情だけ、法人向けで機能だけを語ること

## 10. CTA の圧力設計

### 10-1. 役割の整理

- `無料診断` は、個人向け導線の主CTA
- `無料相談` は、理解や関係構築の後に置く補助CTA
- 外部LPへの遷移は、申込ボタンではなく `内容を理解した人が次へ進む` 導線として扱う
- 法人向けの主CTAは `問い合わせ` または `相談` とする

### 10-2. 配置ルール

- トップでは、個人向けに `無料診断`、法人向けに `法人向けページ / 問い合わせ` を出し分ける
- 個人向けページでは、悩み理解の後に `無料診断` を置く
- `無料相談` は中腹以降、または不安解消パートの後に置く
- 外部LPボタンは、サロン内容、対象者、得られること、参加流れを説明した後に置く

### 10-3. 文言ルール

- 強い煽り文句は使わない
- `今すぐ申し込む` より `まず診断してみる`、`内容を見てから進む` を優先する
- 法人向けでは `お問い合わせはこちら` より、`課題に合わせて相談する` など相談起点の文言が望ましい

## 11. 画面方針メモ

### 11-1. トップ

役割:

- 竹村英子さんが何者かを短時間で伝える
- 個人向け / 法人向けの入口を迷わず選ばせる
- `整える専門家` としての信頼を短時間で立てる

ファーストビュー方針:

- 竹村英子さん本人が主役
- キャッチは、心・体・習慣を整える支援であることがひと目で伝わる
- CTA は `個人の方へ` と `法人・団体の方へ` を並列で見せ、その上で個人向け主導線として `無料診断` を補助表示する

中盤以降の構成方針:

- 個人向け価値
- 法人向け価値
- 実績 / 信頼補強
- 竹村英子さんの背景

の順に、主軸がぶれない形で整理する。

ビジュアル方針:

- 誌面感のあるヒーロー
- 余白をしっかり取った導線ブロック
- 実績はロゴ羅列より、要約された信頼ブロックで見せる

### 11-2. 個人向けトップ

役割:

- 40代以上女性が、自分ごととして読める入口を作る
- 不調や揺らぎを、過剰に不安化せず整理して受け止める
- `じぶん整え習慣サロン` と `無料診断` へ自然につなぐ

ファーストビュー方針:

- 共感しやすい悩みを、静かで落ち着いた言葉で置く
- 強い悩み煽りはしない
- 竹村英子さんの伴走感が伝わる写真を使う

中盤以降の構成方針:

- よくある状態の整理
- `じぶん整え習慣` の考え方
- 体 / 食 / 色の支援軸
- 継続の価値
- 受講者の安心材料
- `無料診断`

の順が基本。

ビジュアル方針:

- あたたかいが甘すぎない
- 写真は寄りを使い、生活接点を感じさせる
- 余白をやや多めに取り、焦らせない画面にする

### 11-3. `じぶん整え習慣とは`

役割:

- 竹村英子さんの支援思想を言語化する
- 単なる健康情報や自己啓発ではないことを伝える
- サロン理解の前提を作る

方針:

- 哲学だけで終わらせず、支援の中身に接続する
- `体養生 × 食養生 × 色養生` の3軸は、図解的に整理して見せる
- 写真と短い要約を組み合わせ、読み疲れない構成にする

### 11-4. `じぶん整え習慣サロン`

役割:

- BtoC 主力商品の説明ページ
- 誰のためのものか、何が得られるか、どう参加するかを明快に伝える

方針:

- 情緒先行ではなく、内容理解と安心を優先する
- `動画101本`、3領域、継続しやすさ、向いている人を具体で見せる
- 申込は急がせず、説明の後で外部LPへ渡す
- `無料診断` や `無料相談` は、参加前の不安解消導線として補助配置する

### 11-5. `無料診断`

役割:

- BtoC の第一入口
- 初めての人が心理的負荷なく接触できる受け皿

方針:

- 診断の意味、所要感、診断後の流れを事前に明示する
- いきなり売り込まれる不安を消す
- 結果が `無料相談` や `サロン` へ自然につながる設計を前提にする

### 11-6. 法人向けトップ

役割:

- 企業人事に `この人に相談できる` と伝える
- 研修や支援が何を改善するのかをわかりやすく示す
- 詳細ページや問い合わせへつなぐ

ファーストビュー方針:

- 課題起点で入りつつ、抽象論で終わらせない
- 竹村英子さん本人の信頼感が出る写真を使う
- `研修メニューを見る` と `相談する` を自然に置く

中盤以降の構成方針:

- 解決できる課題
- 支援の考え方
- 提供メニュー
- 実績 / 導入イメージ
- 問い合わせ

の順を基本にする。

ビジュアル方針:

- BtoC より少し密度高め
- セクションの区切りを明快にする
- 写真は登壇、対話、場づくりの文脈を優先する

## 12. 実績 / お客様の声の見せ方

- 過去の活動量を並べるのではなく、今の主軸につながるものを選ぶ
- 個人向けの声と法人向けの声は混在させず、読み手別に整理する
- 実績は `幅広くやってきた人` ではなく `整える支援を継続してきた人` と見える編集にする
- 法人実績は信頼補強として重要だが、トップの主役は個人向け主力導線を崩さない

## 13. 優先順位

### 優先度 A

- トップ
- 個人向けトップ
- `じぶん整え習慣とは`
- `じぶん整え習慣サロン`
- `無料診断`
- 法人向けトップ

### 優先度 B

- 実績・お客様の声
- 竹村英子について
- 研修メニュー
- 解決できる課題

### 優先度 C

- コラム / お知らせ
- FAQ
- 更新追加される周辺ページ

## 14. 固定領域と更新領域

### 14-1. 固定する部分

- ホーム
- 個人向けトップ
- 法人向けトップ
- `じぶん整え習慣とは`
- `じぶん整え習慣サロン`
- 竹村英子プロフィール
- ブランド説明
- 主要CTAの構造

### 14-2. 更新しやすくする部分

- お知らせ
- コラム
- 実績追加
- 導入事例追加
- 受講者の声追加

## 15. 実装前に守ること

- まだ CSS 設計を確定しない
- まだコンポーネント一覧から入らない
- まだ CMS の詳細設計に入らない
- まだ画面数を増やしすぎない
- まずトップ / 個人向け / 法人向けの視覚原則を崩さない
- 実装都合でトンマナを均質なテンプレートに落とさない
- 更新性を理由に、固定すべきブランド表現まで可変にしない
